# Safety OS — AI 기반 EHS 플랫폼

> Claude Code Harness Engineering 기반 대한민국 EHS 규정 준수 자동화

## 개요

Safety OS는 대한민국 EHS(환경·보건·안전) 분야를 위한 AI 에이전트 기반 안전 운영 시스템입니다. 법적으로 요구되는 감사 증거 기록을 유지하면서 핵심 안전 업무 워크플로우를 자동화합니다.

## 법적 근거

| 법령 | 범위 | 시행일 |
|------|------|--------|
| 산업안전보건법 | 주요 EHS 프레임워크 | 1990년 (2023년 개정) |
| 중대재해처벌법 | 중대산업재해 형사책임 | 2022-01-27 |

## 아키텍처

```
PM (최고안전보건책임자, CSO)
├── 안전거버넌스매니저 (SGM) — 전략
│     KPI 설정, 정책 승인, 법규 모니터링
└── 안전워크플로우매니저 (SWM) — 실행
      ├── 컴플라이언스 에이전트
      ├── 위험성평가 에이전트
      ├── 비상대응 에이전트
      └── 감사 에이전트
```
