# Data Center Daily Workflows

| Workflow | Description | Risk |
|----------|-------------|------|
| Facility Rounds | General facility and physical security checks | Low |
| Power System Maintenance | UPS, generator, and PDU inspections | High |
| Cooling System Maintenance | HVAC, CRAC, and chiller checks | Medium |
| Fire Suppression Systems | VESDA, gas suppression system status | High |
| Server Rack Installation | Physical handling, grounding, power connection | Medium |
| Emergency Response Drill | Drill for power outage or fire scenarios | Medium |
