# tbm-tool-box-meeting Workflow

## 1. Objective
TBM (Tool Box Meeting) — 일일 작업 전 안전회보. 한국 건설업 고유 관행.

## 2. TBM 내용
1. 전일 안전사고/문제점 공유
2. 금일 작업별 위해 요소 교육
3. 안전장구 점검 (안전모/안전대/안전화)
4. 작업별 안전수칙 확인
5. 건의사항 수렴

## 3. Workflow Steps
1. **작업 전 15분 실시**: 작업 시작 전
2. **참석 체크**: 전원 서명
3. **주제 기록**: 일지 작성
4. **사진 기록**: 교육 실시 증빙
5. **결근자 follow-up**: 별도 교육 실시

## 4. Evidence Record
Generate `ehsconst-tbm-record.json` with 참석자 명단.
