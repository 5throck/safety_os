# subcontractor-management Workflow

## 1. Objective
협력업체 (하도급자) 안전 관리 — 다층 하도급 구조 대응.

## 2. 하도급 안전 의무 (SAPA Article 13)
- 원도급자: 하도급자 안전관리 지도/감독
- 하도급자: 자체 안전관리의무 이행
- 발주처: 원도급자 선정 기준에 안전이력 반영

## 3. Workflow Steps
1. **하도급자 사전 심사**: 안전이력, 안전관리체계, 안전관리자 자격.
2. **안전협약 체결**: 안전관리비, 안전의무, 보고 체계.
3. **작업 전 안전교육**: 원도급자가 실시.
4. **작업 중 감독**: 일일 순찰, 점검회의.
5. **안전이력 관리**: 사고/위반 기록, 차후 심사 반영.
