# qau-inspection Workflow

## 1. Objective
Conduct independent QAU inspections per OECD GLP Section 3. **GLP-unique workflow** — QAU independence is fundamental.

## 2. Applicability
All GLP-regulated facilities. QAU must be organizationally independent from study teams.

## 3. Inspection Types
- **Facility inspection**: Annual comprehensive audit
- **Study-based inspection**: Per major study milestones
- **Process-based inspection**: Recurring operations (e.g., calibration)
- **For-cause inspection**: Triggered by deviation/complaint

## 4. Workflow Steps
1. **Annual inspection plan**: Risk-based prioritization.
2. **Inspection conduct**: Per SOP, with documentation.
3. **Findings classification**: Critical / Major / Minor.
4. **Report to Study Director**: Findings with proposed CAPA.
5. **CAPA verification**: Follow-up inspection.
6. **Closure**: QAU sign-off after effectiveness verification.
7. **Management reporting**: Status to Test Facility Management.

## 5. QAU Independence
QAU personnel must NOT participate in study execution. `glp-agent` provides QAU functional support but does NOT replace organizational independence.

## 6. Evidence Record
Generate `glp-qau-inspection-record.json` with inspection type, findings, CAPA status.
