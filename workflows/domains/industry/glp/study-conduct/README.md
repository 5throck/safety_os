# study-conduct Workflow

## 1. Objective
Execute GLP study per approved protocol with full data integrity per OECD GLP Section 9.

## 2. Applicability
All study activities from initiation to completion.

## 3. Workflow Steps
1. **Pre-study verification**: Protocol approved, equipment calibrated, personnel trained.
2. **Test system receipt**: Animals/cells received, acclimatized, health-checked.
3. **Study initiation**: Per protocol start date.
4. **Daily observations**: Recorded per SOP, ALCOA+ compliance.
5. **In-life phase**: Dosing, observations, sampling per protocol.
6. **Terminal phase**: Necropsy, sample collection, lab analysis.
7. **Raw data generation**: All entries attributable, contemporaneous.
8. **Study completion**: Final report initiated.

## 4. Evidence Record
Generate `glp-study-conduct-record.json` with daily log, deviations, raw data references.
