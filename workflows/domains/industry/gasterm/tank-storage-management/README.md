# 저장탱크 운영 관리 (Tank Storage Management) Workflow

## 1. 목적
본 워크플로우는 한국 가스안전 관련 법령(고압가스안전관리법 (High-Pressure Gas Safety Control Act) 제14조, 위험물안전관리법 (Dangerous Substances Safety Management Act))에 따른 저장탱크 운영 관리 업무를 관리한다.

## 2. 워크플로우 단계
1. 사전 작업 점검
2. 실행
3. 작업 후 검증
4. 문서화 및 감사 추적

## 3. 증거 기록
다중 출처의 `legal_basis`를 포함하여 `gasterm-tank-storage-record.json`을 생성한다.

## 4. 법적 면책 고지
> 본 시스템은 워크플로우 자동화 지원만 제공하며, 최종 판단은 자격을 갖춘 가스안전관리자의 검토가 필요하다.
