# 일일 안전점검 (Daily Safety Inspection) Workflow

## 1. 목적
일일 안전점검 — 매일 작업 전 현장 안전 상태 확인.

## 2. 워크플로우 단계 (Workflow Steps)
1. **작업 전 점검**: 안전관리자/안전감리가 현장 순찰.
2. **위해 요소 식별**: 추락/붕괴/전기/화재 등.
3. **시정 조치**: 즉시 조치 가능 항목 vs 작업 중지 필요.
4. **점검 기록**: 사진 포함, 서명.
5. **TBM 연계**: 점검 결과 TBM 회보에 반영.

## 3. 증거 기록 (Evidence Record)
`ehsconst-inspection-record.json` 생성.
