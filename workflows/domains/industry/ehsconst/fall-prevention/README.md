# fall-prevention Workflow

## 1. Objective
추락 방지 — 한국 건설 사망사고 원인 1위 (~42%).

## 2. Fall Prevention Hierarchy
1. **Elimination**: 작업 자체 제거 (설계 변경)
2. **Passive fall protection**: 안전난간, 추락방지망
3. **Active fall protection**: 안전대, 활동제한장치
4. **Administrative**: 작업허가서, 교육
5. **PPE**: 안전모, 안전화 (최후 수단)

## 3. Workflow Steps
1. **추락 위험 평가**: 작업 높이, 기간, 작업자 수.
2. **방호조치 선정**: 위계에 따른 우선 적용.
3. **설치 및 점검**: 안전장비 적정 설치 확인.
4. **작업자 교육**: 추락방지장치 사용법.
5. **구조 계획**: 추락 시 신속 구조 절차.
