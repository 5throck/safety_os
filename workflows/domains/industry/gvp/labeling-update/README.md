# labeling-update Workflow

## 1. Objective
Update safety labeling based on PV findings per EU GVP Module 15.

## 2. Label Components
- Contraindications
- Warnings and precautions
- Adverse reactions (frequency, severity)
- Special populations (pregnancy, pediatric, elderly)
- Drug interactions

## 3. Workflow Steps
1. **Trigger**: New signal, MFDS request, foreign labeling update.
2. **Data compilation**: ICSR analysis, clinical evidence.
3. **Drafting revision**: Per Korean labeling guidelines.
4. **Internal review**: Medical, Regulatory, Legal.
5. **MFDS submission**: Protocol variation.
6. **Implementation**: Updated labeling distribution.
7. **DHCP notification**: Dear Healthcare Professional letters if urgent.

## 4. Evidence Record
Generate `gvp-labeling-record.json` with change details.
