# gdp-self-inspection Workflow

## 1. Objective
Conduct GDP self-inspection per KGDP, identify non-conformances, implement CAPAs. Pattern reused from GMP self-inspection.

## 2. Applicability
All GDP-regulated operations at pharmaceutical distribution facilities.

## 3. Frequency
- Default: annual (KGDP minimum)
- High-risk areas (cold chain, DTS, hazardous drugs): semi-annual

## 4. Workflow Steps
1. Annual self-inspection plan with risk-based prioritization.
2. Pre-inspection checklist preparation.
3. Conduct inspection with documentation.
4. Findings classification (critical/major/minor).
5. CAPA plan via deviation tracking.
6. Management review and resource commitment.
7. Follow-up and effectiveness check.
8. Closure and 5-year retention.

## 5. Evidence Record
Generate `gdp-self-inspection-record.json` with `frequency.risk_based_adjustment`.

## 6. KPI
- 100% on-schedule completion
- 0 CAPAs overdue >90 days
