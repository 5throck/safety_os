# participant-enrollment Workflow

## 1. Objective
Participant enrollment per protocol inclusion/exclusion criteria.

## 2. Workflow Steps
1. **Screening**: Medical history, demographics, eligibility criteria.
2. **Inclusion/exclusion verification**: Per protocol, no waiver.
3. **Informed consent verification**: ICF signed before any procedure.
4. **Enrollment**: Subject ID assignment, registration.
5. **Randomization** (if applicable): Per ICH E9(R1).
6. **Withdrawal tracking**: Reasons, IP stop decisions.

## 3. Vulnerable Population Protections
- Minors: Parent/guardian + assent
- Pregnant women: Additional IRB review
- Prisoners: Limited research per 생명윤리법

## 4. Evidence Record
Generate `gcp-participant-record.json`.
