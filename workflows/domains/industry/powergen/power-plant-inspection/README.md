# 발전설비 정기 점검 (Power Plant Periodic Inspection) Workflow

## 1. 목적
본 워크플로우는 한국 발전 안전 관련 법령에 따라 발전설비 정기 점검을 수행한다. 주요 근거법령은 전기사업법 (Electric Utility Act) 제47조 및 전기안전관리법 (Electrical Safety Control Act) 제16조이다.

## 2. 워크플로우 단계
1. 작업 전 점검
2. 안전조치 적용 작업 수행
3. 작업 후 확인
4. 문서화 및 감사추적 (audit trail)

## 3. 증거기록 (Evidence Record)
다중 출처의 `legal_basis`를 포함하여 `powergen-inspection-record.json`을 생성한다.

## 4. 법적 면책 고지
> 본 시스템은 워크플로우 자동화 보조만 제공하며 법적 자문이 아니다. 최종 결정은 자격을 갖춘 전기·설비 안전관리자의 판단이 필요하다.
