# 발전설비 정기 점검 (Power Plant Periodic Inspection) Workflow

## 1. Objective
This workflow manages Power Plant Periodic Inspection per Korean power generation safety regulations.

## 2. Workflow Steps
1. Pre-operation check
2. Execution with safety controls
3. Post-operation verification
4. Documentation and audit trail

## 3. Evidence Record
Generate `powergen-inspection-record.json` with multi-source legal_basis.

## 4. Legal Disclaimer
> Workflow automation only. Final decisions require qualified electrical/plant safety manager.
