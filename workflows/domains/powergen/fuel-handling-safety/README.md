# 연료 취급 안전 (Fuel Handling Safety) Workflow

## 1. Objective
This workflow manages Fuel Handling Safety per Korean power generation safety regulations.

## 2. Workflow Steps
1. Pre-operation check
2. Execution with safety controls
3. Post-operation verification
4. Documentation and audit trail

## 3. Evidence Record
Generate `powergen-fuel-handling-record.json` with multi-source legal_basis.

## 4. Legal Disclaimer
