# monitoring-visits Workflow

## 1. Objective
CRA (Clinical Research Associate) monitoring visits per ICH E6(R3) Section 8.

## 2. Visit Types
- **Site Initiation Visit (SIV)**: Before enrollment
- **Routine Monitoring Visits**: Per monitoring plan
- **For-cause visits**: Triggered by issues
- **Close-out Visit (COV)**: After database lock

## 3. Workflow Steps
1. **Pre-visit preparation**: Source documents identification.
2. **On-site review**: SDV, ICF verification, IP accountability.
3. **Findings documentation**: Per monitoring SOP.
4. **Follow-up letter**: Within 7 days to investigator.
5. **CAPA tracking**: Until resolution.
6. **Visit report**: Submitted to Sponsor.

## 4. Risk-Based Monitoring
ICH E6(R2/R3) encourages risk-based approach — critical data/effect parameters get higher SDV coverage.

## 5. Evidence Record
Generate `gcp-monitoring-record.json`.
