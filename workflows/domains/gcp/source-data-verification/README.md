# source-data-verification Workflow

## 1. Objective
Source Data Verification (SDV) per ICH E6(R3) Section 9 with ALCOA+ integrity.

## 2. ALCOA+ Principles
- Attributable, Legible, Contemporaneous, Original, Accurate
- + Complete, Consistent, Enduring, Available

## 3. Source Documents
- Medical records, lab reports, ECGs
- Investigator notes, participant diaries
- ICFs, randomization records
- IP accountability logs

## 4. SDV Coverage
- 100% for primary efficacy endpoints
- 100% for safety data (SAEs)
- Risk-based for secondary endpoints

## 5. Workflow Steps
1. **Source identification**: Per site SOP.
2. **SDV execution**: CRA cross-checks CRF vs source.
3. **Discrepancy tracking**: Query resolution.
4. **Documentation**: SDV completion in monitoring record.

## 6. Evidence Record
Generate `gcp-source-data-record.json` with ALCOA+ compliance object.
