# irb-review Workflow

## 1. Objective
IRB (생명윤리심의위원회) review per KGCP + 생명윤리법 + ICH E6(R3) Section 4.

## 2. Review Types
- **심의 (full review)**: New trial, major amendments
- **신속심의 (expedited)**: Minor changes, minimal risk
- **면제 (exempt)**: Below risk threshold

## 3. Workflow Steps
1. **Submission package**: Protocol, ICF, IB, investigator qualifications.
2. **IRB review**: Independent committee evaluation.
3. **Decision**: Approved / Conditional / Rejected.
4. **Conditions**: Address within defined timeline.
5. **Continuing review**: Annual minimum for ongoing trials.

## 4. Independence
IRB must be organizationally independent of sponsor and investigator. MFDS-certified IRB required.

## 5. Evidence Record
Generate `gcp-irb-approval-record.json`.
