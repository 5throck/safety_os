# equipment-calibration Workflow

## 1. Objective
Maintain equipment calibration and qualification per OECD GLP Section 5. Pattern reused from GMP equipment-qualification.

## 2. Applicability
All equipment used in GLP studies (balances, analyzers, environmental chambers).

## 3. Workflow Steps
1. **Equipment inventory**: Each unit with unique ID.
2. **IQ/OQ/PQ**: Installation, Operational, Performance qualification.
3. **Calibration schedule**: Per manufacturer + SOP.
4. **Calibration execution**: Per ISO 17025 traceable standards.
5. **Maintenance records**: Preventive and corrective.
6. **Out-of-tolerance investigation**: If calibration fails, assess study impact.
7. **Decommissioning**: Safe disposal with data retention.

## 4. Cross-Domain
Pattern reused from GMP `equipment-qualification` workflow.

## 5. Evidence Record
Generate `glp-equipment-record.json` with calibration history.
