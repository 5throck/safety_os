# informed-consent Workflow

## 1. Objective
Informed Consent Form (ICF) management per 생명윤리법 + ICH E6(R3) Section 5.

## 2. Workflow Steps
1. **ICF drafting**: Per protocol, plain language.
2. **IRB approval of ICF version**.
3. **Translation**: Korean + other applicable languages.
4. **Consent process**: Private setting, sufficient time, witness for vulnerable populations.
5. **Version control**: Re-consent when ICF changes.
6. **Documentation**: Signed/dated ICF in source documents.

## 3. Key Elements (Helsinki Declaration)
- Purpose, procedures, duration
- Risks, discomforts, alternatives
- Confidentiality, compensation, contact info
- Voluntary participation, withdrawal right

## 4. Evidence Record
Generate `gcp-informed-consent-record.json` with ICF version.
