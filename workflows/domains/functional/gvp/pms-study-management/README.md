# pms-study-management Workflow

## 1. Objective
Manage Korean Post-Marketing Surveillance (PMS) studies — mandatory for new drugs 6-8 years post-approval.

## 2. Korean PMS Specifics
- Mandatory for first 6-8 years post-approval (new drugs)
- Population-based Korean study design
- Results feed into drug re-evaluation (재평가)
- Korean Standard Drug Code (KDC) tracked

## 3. Workflow Steps
1. **Study design**: Protocol development (Korean population considerations).
2. **Site selection**: Korean hospitals/clinics.
3. **Enrollment**: Per inclusion criteria.
4. **Follow-up**: Long-term (typically 1-3 years).
5. **Data collection**: ADRs, effectiveness outcomes.
6. **Interim analysis**: Per protocol.
7. **Final report**: Submission to MFDS for re-evaluation.

## 4. Evidence Record
Generate `gvp-pms-record.json` with study enrollment and outcome data.
