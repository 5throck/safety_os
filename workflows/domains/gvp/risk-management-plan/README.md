# risk-management-plan Workflow

## 1. Objective
Maintain Risk Management Plan (RMP) per ICH E2E and EU GVP Module 5.

## 2. RMP Components
- Safety Specification: epidemiology, non-clinical, clinical, post-market
- Pharmacovigilance Plan: routine + additional monitoring
- Risk Minimization Plan: educational materials, restrictions, DHCP letters

## 3. Workflow Steps
1. **Safety specification**: Identify important identified/potential/missing risks.
2. **PV plan**: Define additional monitoring needs.
3. **Risk minimization**: Educational materials, restricted access programs.
4. **RMP version control**: Update per signal events, MFDS review.
5. **Implementation tracking**: PV plan execution metrics.
6. **Effectiveness assessment**: Periodic (annually) evaluation.

## 4. Evidence Record
Generate `gvp-rmp-record.json` with `rmp_version_ref` field.
