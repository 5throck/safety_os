# mechanical-accident Workflow

## 1. Objective
기계 사고 (끼임/깔림/절단) 대응.

## 2. 주요 기계 사고
- 프레스/컨베이어 끼임
- 중기 깔림
- 회전 기계 감김
- 절단 기계 부상

## 3. 응급 대응
1. 기계 정지 (비상 정지 버튼)
2. 전원 차단
3. 피해자 구출 (절단/인양)
4. 지혈/응급 처치
5. 병원 이송 (절단 부위 동반)

## 4. 사고 현장 보존
- 기계 상태 기록 (사진/동영상)
- 작업 상황 서면 기록
- 목격자 진술
- 안전관리자 즉시 보고

## 5. 예방 (중요)
- 안전장치 (가드/라이트커튼)
- 정기 점검
- 작업 교육
- 비상 정지 버튼 접근성

## 6. Evidence Record
Generate `emergency-mechanical-record.json`.
