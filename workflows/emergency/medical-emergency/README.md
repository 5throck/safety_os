# medical-emergency Workflow

## 1. Objective
현장 의료 응급 상황 대응 (심정사/중증 알레르기/중증 부상).

## 2. 주요 시나리오
- **심정사 (Sudden Cardiac Arrest)**: AED 사용, CPR
- **중증 알레르기 반응**: 아나필락시스 — 에피네프린
- **중증 화상/절단**: 지혈/압박/응급 이송
- **열사병/저체온증**: 체온 조절
- **중독 (화학물질/일산화탄소)**: 해독제/산소

## 3. Workflow Steps
1. 신고 접수 (119 연락)
2. 초기 응급 처치 (현장)
3. AED/구급함 사용
4. 병원 이송 (구급차)
5. 사후 보고 및 사고 조사

## 4. 시설 요건 (산업안전보건법 Article 13)
- 구급설비 비치 (상비약/AED/구급함)
- 응급처치 교육 이수자
- 119 직통 전화
