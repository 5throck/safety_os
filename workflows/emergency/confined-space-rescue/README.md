# confined-space-rescue Workflow

## 1. Objective
밀폐공간 사고 시 신속 구조. 사망사고 고빈도 (질식/추락).

## 2. 주요 밀폐공간
- 맨홀/수처리장
- 저장 탱크/실로
- 터널/굴착면
- 보일러/연도

## 3. 사고 유형
- 질식 (산소 부족/독가스)
- 중독 (황화수소/일산화탄소)
- 추락
- 끼임

## 4. 구조 절차
1. 구조 대기 (절대 진입 금지)
2. 가스 농도 측정 (O2/H2S/CO/LEL)
3. 환기 (강제 송풍)
4. PPE 착용 구조대 진입
5. 외부에서 호스/삼각대 사용

## 5. 비상 장비
- 3각대/호이스트
- 4가지 가스 측정기
- 자급식 호흡기 (SCBA)
- 구조용 호스

## 6. Evidence Record
Generate `emergency-rescue-record.json`.
