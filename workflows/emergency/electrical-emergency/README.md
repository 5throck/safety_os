# electrical-emergency Workflow

## 1. Objective
감전 사고 및 전기 설비 사고 대응.

## 2. 사고 유형
- 감전 (접촉/아크)
- 활선 작업 사고
- 누전 사고
- 정전 사태

## 3. 응급 대응 원칙
- **전원 차단 최우선** (감전자 접촉 금지)
- 절연 보호구 착용
- 소화 시 CO2/분말 (물 사용 금지)

## 4. Workflow Steps
1. 전원 차단 (차단기/메인)
2. 감전자 격리 (절연물 사용)
3. CPR (심정사 동반 시)
4. 응급 이송
5. 누전 원인 조사

## 6. 예방
- 정기 절연 저항 측정
- 누전 차단기 설치
- 활선 작업 시 허가서
- 접지 확인

## 7. Evidence Record
Generate `emergency-electrical-record.json`.
