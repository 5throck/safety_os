# high-angle-rescue Workflow

## 1. Objective
고소 작업 사고 (추락/현수) 시 신속 구조.

## 2. 사고 시나리오
- 안전대 현수 (추락 후 매달림 — Suspension Trauma 위험)
- 크레인/고소작업대 고장
- 건축물 외벽 고립
- 송전탑/풍력터빈 고립

## 3. Suspension Trauma (현수성 실신)
- 매달린 상태 5-10분 내 실신 가능
- 혈액 순환 장애
- 신속 구조 필수

## 4. 구조 장비
- 고소작업대 (Aerial Lift)
- 로프/하네스
- 3각대/인양기
- 헬기 (극한 상황)

## 5. Workflow Steps
1. 사고 인지 및 구조대 출동
2. 피해자 상태 확인 (의식/호흡)
3. 안전한 구조 경로 설정
4. 구조 실행
5. 응급 처치/이송
