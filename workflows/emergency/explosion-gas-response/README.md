# explosion-gas-response Workflow

## 1. Objective
폭발 및 가스 누출 사고 대응.

## 2. 주요 원인
- 가스 누출 (LPG/LNG/수소)
- 정전기/불꽃
- 화학 반응 (중합/분해)
- 밀폐 용기 과압

## 3. Workflow Steps
1. 폭발/누출 감지
2. 즉시 대피 (반경 500m 이상)
3. 가스 농도 측정 (LEL 10% 이상 시 대피)
4. 화원 제거/차단
5. 소방서 신고 (119)
6. 환기/세정 후 재진입

## 4. 예방 (중요)
- 가스 검지기 설치
- 정전기 제거
- 화기작업 엄격 통제
- 압력 용기 정기 검사
